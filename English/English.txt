Reading strategies

Verb patterns

1. verb + gerund
Para estos verbos, el verbo que viene a continuación debe ir en gerundio
Ex: sending, coding, uploading, programming, seeing, giving, complaining, enjoy, keep, mind, finish, practice,
suggest, recommend...
Ex: I really enjoyed learning about programming languages.

2. verb + infinitive
Para estos verbos, el verbo que le sigue debe ir con to
Ex: to drive, to see, to create, to copy, to do, to deliver, to download, to think, want, learn, promise, propose,
decide, pretend, agree...
Ex: I want to do exercise tonigth.

3. verb + infinitive or gerund 
Para estos verbos, el verbo que le sigue puede ir con "to" o con "ing" y no cambia el significado
Ex: have, go, leave, watch, fix, clean, play, work, start, continue, begin, like, hate, love, prefer
Mario started to learn German last month.

4. verb + infinitive or gerund
Para estos verbos, el verbo que le sigue puede ir con "to" o con "ing" pero esto si cambia el significado
Ex: look, try, speak, forgive, take, plan, stop, type, remember, quit, forget, regret.

I stopped using Java since it is pretty complex
ans: I don't use Java anymore
I stopped to have lunch on the way home
ans: We interrupted our travel to have lunch
Para los ejemplos con stop se debe tener presente que si uso "ing" estoy diciendo que pare determinada actividad de
forma definitiva, mientras que si uso "to" estoy diciendo que para determinada actividad para realizar otra.

the bus stopped to pick up people / el autobús se detuvo para recoger gente
the bus stopped picking up people / el autobús dejó de recoger gente

you try to repair this computer / intenta reparar esta computadora (complicated/difficult)
you need to try wearing this jacket / tienes que intentar usar esta chaqueta (the thing is not difficult)
El try seguido de "to" se usa para hablar de acciones complicadas, pero seguido de "ing" se usa para cosas sencillas

I remembered to buy some milk (recuerdo cercano de acción que puedo ejecutar)
I remembered playing when I was a child (recuerdo de remembranzas del pasado)

she forgot to lock the door (olvido momentaneo)
she forgot programming in Java (olvido de recuerdos o aprendizajes pasados)

Verb patterns

Después de los siguientes verbos debe ir "to" =>
(verb) + to + verb = [afford, agree, ask, choose, decide, expect, forget, help, hope, learn, need, offer, plan, 
prepare, pretend, promise, refuse, seem, want, manage, would like]

Después de los siguientes verbos, el verbo que le sigue debe ir con "ing" =>
(verb) + verb-ing = [consider, dislike, enjoy, discuss, finish, imagine, keep, mind, spend(time), suggest, avoid,
admit, miss, practice, can't help(solo si va acompañado de can't)]

Reading strategies

Prefixes and suffixes

Prefixes

Se adicionan al principio de la palabra base 

un(prefix) happy(root) => unhappy(new word) not happy 
Los prefijos no cambian la escritura de la palabra

re: again or back = [REstructure, REvisit, REappear, REbuild, REfinance]
over: too much = [OVERbook, OVERsleep, OVERwork]
out: more or better than others = [OUTperform, OUTbid]

use of hyphenate(-) Prefixes

rules

1. When adding a prefix to a proper noun
Ex: un-American, pre-Columbian, mid-August.

2. Cuando el prefijo termina en la misma letra con la que inicia la palabra que vamos a modificar uso el (-)
Ex: re-entry, anti-icing, re-educate

3. special Prefixes
Luego de ex and self siempre se usa (-)
Ex: self-conscious, ex-student, ex-president

4. Clarification: cuando necesitamos que la palabra quede con el sentido que queremos darle
Ex: the mother reminder the children to re-cover the sandbox after were done playing

Prefijos direrentes con significado similar 
un- => in- and non- significan no o opuesto de  
mis- => and ir- significan equivocado o incorrecto

se pueden formar palabras con letras dobles como
im- => immeasurable

Suffixes

Se adicionan al final de la palabra base, lo que cambia el significado y la escritura

flavor => flavorLESS (sabor/sin sabor)

También se clasifican como sufijos = [-er, -s, -es, -ed, -ing and -ly]
